cPanel Deployment Gateway

Static Export is Configured.

Next.js is configured for static export. Follow the simple steps below on your local machine to build the site and copy the files to cPanel.

Deployment Steps

01

1. Run local build

Run the build command on your local machine to export all pages as static HTML, CSS, and Javascript files.

npm run build
02

2. Locate "out" directory

Once the build finishes successfully, Next.js generates a folder named out in your project root containing your index.html.

03

3. Upload to cPanel

Compress the contents of the out folder into a ZIP file, upload it to cPanel File Manager inside public_html, and extract it.
